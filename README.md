# 🎨 Design Portfolio

A modern and responsive **design portfolio website** showcasing my creative work including posters, social media creatives, and motion graphics (reels).

---

## 🚀 Features

- ✨ Clean and minimal UI  
- 📱 Fully responsive design  
- 🎬 Video reel previews (autoplay & loop)  
- 🗂️ Category-based filtering (Posters / Social Media / Reels)  
- 🎯 Smooth hover effects and overlays  

---

## 🛠️ Tech Stack

- HTML5  
- CSS3  
- JavaScript
